ADropdown
=========

Angular directive for create dropdowns

Crate any element and add directive dropdown

Example
---
```html
<i dropdown></i>
<ul class="menu">
    <li>
        <a href="#">option 1</a>
    </li>
    <li>
        <a href="#">option 2</a>
    </li>
</ul>


<button dropdown></button>
<ul class="menu">
    <li>
        <a href="#">option 1</a>
    </li>
    <li>
        <a href="#">option 2</a>
    </li>
</ul>
```
